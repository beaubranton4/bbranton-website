## Conventions

- One file per topic thread. Add new threads as new files; update existing ones if a conversation revisits the same idea.
- Lead with **Verdict** (good/bad/it depends + one-sentence why).
- Then **Why** (market signal, search data, competitor landscape).
- Then **Open questions** (what would change the verdict).
- Append date-stamped entries when conversations revisit a topic.
