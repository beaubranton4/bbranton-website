# Manifestos

External essays, frameworks, and points of view worth keeping around — summarized faithfully with source links. Distinct from [explorations/](../explorations/) (which is *Beau's own* thinking on specific ideas) and from [explorations/frameworks.md](../explorations/frameworks.md) (which is *Beau's synthesized* generative lenses).

**Convention:** one file per essay/article. Lead with the source link + author + date, then a one-line takeaway, then a faithful summary. Add a short "how this connects to my board" section at the end if useful, clearly separated from the summary.
